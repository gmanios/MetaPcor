# MetaPcor
Meta-analysis of gene expression studies using partial correlation as effect size

Web version of MetaPcor available at : http://rs.dib.uth.gr:3839/metapcor/metapcor2023/


## Requirements
* GEOquery
* corpcor
* space
* data.table
* GeneNet
* meta
* metafor
* dplyr
* tidyverse
* purrr
* DescTools
* ggplot2
* ff
* matrixcalc
* gprofiler2
* visNetwork
* geomnet
* igraph
* plotly
* readxl
